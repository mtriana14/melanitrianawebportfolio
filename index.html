Simple Facial Recognition

In November 2024, I developed a facial recognition application using Python and OpenCV, designed to identify and verify individuals based on facial features. The application captures and processes real-time video input, detecting faces and drawing bounding boxes around them for clear visualization. By leveraging OpenCV’s robust image-processing capabilities, I ensured accurate and efficient detection.

The project was built and tested in VS Code, with a focus on performance optimization and user-friendly feedback, allowing for quick and responsive recognition.

Tictactoe Simple Website

This is a simple website that uses AI to run against a player.

In February 2024 to March 2024, I developed an interactive Tic-Tac-Toe game as a web application using JavaScript, CSS, and HTML. The game logic was implemented in JavaScript, handling **player turns, win conditions, and tie scenarios** to ensure smooth and fair gameplay. CSS was used to style the interface, creating a responsive and visually appealing design. The project was built and tested in VS Code, with version control utilized to manage and track development progress effectively..
